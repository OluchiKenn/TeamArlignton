## How to Setup and Run

1. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

2. Run the application:
   ```
   python3 run.py
   ```

3. Open your web browser and go to:
   ```
   http://127.0.0.1:5000
   ```

That's it! The app will open on the login page.
